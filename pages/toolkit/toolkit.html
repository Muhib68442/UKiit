How these CMD Files Works on your PC :

  • 1.Deletor Batch File will delete your system's
    • temp (deletes temporary files)
    • %temp% (deletes advanced temporary files)
    • prefetch (deletes cache files)
    • recent (delete shortcut copy of recent files)
    • Open Disk Cleanup (To clean more cache and junk files)
    • Open Settings>System>Storage (to delete more temporary files)
    • tree (Run a tree command to free up directories)
  • 2.Defrag Batch File will defrag your system's storage :
    • Analyze Storage
    • Defrag HDD Files
    • Trim SSD Files
